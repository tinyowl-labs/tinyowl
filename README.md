# TinyOwl

SvelteKit frontend for the TinyOwl archaeological data platform.

## Developing

```sh
npm install
npm run dev
```

Requires `tinyowl-server` on port 8080 (Vite proxies `/api/v1` and `/media`).

## Building

```sh
npm run build
```

## Docs

In-app docs live under `src/lib/docs/` and are served at `/docs`.
